   1  # phpsvnclient #
   2  
   3  **phpsvnclient** is a class that can perform read operations from a SVN server
   4  (over WebDAV).
   5  
   6  It can be used to retrieve files from an SVN repository in pure PHP, thus
   7  without using separate programs or the need to have SVN support within PHP.
   8  
   9  ## Details ##
  10  
  11  It accesses a given remote SVN repository using the WebDAV protocol and
  12  performs several types of operations:
  13  
  14  * List all files in a given SVN repository directory
  15  * Retrieve a given revision of a file
  16  * Retrieve the log of changes made in a repository or in a given file between
  17    two revisions
  18  * Get the repository's latest revision

  39  ## Examples ##
  40  
  41  Fetch file contents in SVN repository:
  42  
  43      <?php
  44      require_once  'phpsvnclient.php';
  45      $phpsvnclient = new phpsvnclient('http://phpsvnclient.googlecode.com/svn/');
  46      $file_content = $phpsvnclient->getFile('trunk/phpsvnclient.php');
  47      echo $file_content;
  48      ?>
  49  
  50  Dump raw SVN description information about a folder:
  51  
  52      <?php
  53      require_once  'phpsvnclient.php';
  54      $phpsvnclient = new phpsvnclient('http://phpsvnclient.googlecode.com/svn/');
  55      $raw_dump = $phpsvnclient->rawDirectoryDump('/trunk/');
  56      print_r($raw_dump);
  57      ?>
