  34  2010-04-04 : jQuery Hintbox 1.4 released (Easter release). From this function jQuery 1.4 will be used.
  35  - New event 'onListRetrieved' added. See examples and documentation to learn more about it.
  36  - ImmediateList option added. Thans to jaco-pixeldump for the code :-)
  37  - Fix not required full list caching bug
  38  - Minor bugs fixed

  41  2009-08-18 : jQuery Hintbox 1.3 released
  42  - MatchRule option added. This option is enabled when MatchHint options is enabled too. 
  43    It allows to filter result list values by 3 criteria: StartsWith [S], EndsWith [E], Contains, [C].
  44    Values allowed are "S", "E", "C".
  45    Default value is: "S".
  46  - JSON option added. If json option is enabled, hintbox plugin want to receive a list made like this:
  47  
  48    {
  49      "list" : [
  50        "AAA",
  51        "BBB",
  52        "CCC"
  53      ]
  54    }
  55  
  56    where "AAA", "BBB", "CCC" are the list values.
  57  - Minor code optimizations and fixes
  58  
  59  
  60  2009-05-16 : jQuery Hintbox 1.2 released
  61  - Match hint option added. This option allows to filter resulting list showing only entries matching with input text. Default value is -> matchHint : false
  62  - Disabled automatic browser autocomplete function. Thanks Eric ;-)
  63  - Fix css bug when resulting list was empty
  64  
  65  
  66  2009-04-17 : jQuery Hintbox 1.1 released
  67  - Sort option added. It implements 'Levenshtein distance' (http://en.wikipedia.org/wiki/Levenshtein_distance). Default value is -> sort : false
  68  - UseCache option added. Default value is -> useCache : true
  69  - Limit option added. If limit value is <= 0 resulting list has no limit. Default value is -> limit: 0

  72  2009-03-31 : jQuery Hintbox 1.0 released
